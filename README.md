# poke-api-consume
poke_api_consume

# dependencies
- npm

# install jest:


```
npm install -g typescript
npm install -g ts-node
npm install -g jest
npm install express@^4.18.2
npm install @types/jest --save-dev
npm install ts-jest --save-dev
```
## configure
```
npm init
npx tsc --init
```

## run tests
`jest --forceExit`

## run script
`ts-node ./src/pokemon.ts`
